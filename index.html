Tecnologias

Angular 1/2

Mantido pela Google, é um Framework MVVM para estruturar aplicações Web modularizadas e concisas.

Ionic 1/2

Junto ao em Angular, Ionic oferece suporte visual e funcional para desenvolver aplicações híbridas de qualidade.

Create JS

Biblioteca de animações em canvas com ênfase em performance. Utilizado no marketing digital e no desenvolvimento de games.

Construct 2

Engine 2D para jogos HTML5, apresenta um desenvolvimento fluído possibilitando a criação de jogos leves e de qualidade para desktop, mobile, Wii U e Xbox.

MongoDB

Banco de Dados orientado à documentos, possibilita a rápida manutenção, alta escalabilidade e buscas performáticas.

NodeJS

Plataforma JavaScript, frequentemente utilizada para o desenvolvimento de Servidores Web com o framework Express.

Polymer

Framework baseado em WebComponents para desenvolvimento de aplicações Web e híbridas eficientes.

Android Java

Desenvolvimento Android nativo tendo acesso às API's do sistema como Bluetooth, Câmera, comunicação USB OTG, etc.

Arduino

Prototipagem eletrônica para desenvolvimento de soluções em automação e Internet das Coisas.

Node Webkit

Desenvolvimento de aplicações Desktop crossplatform utilizando da plataforma NodeJS, possibilitando a manipulação de funcionalidades nativas do sistema como manipulação de arquivos e processos.
